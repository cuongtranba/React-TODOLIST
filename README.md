# React-TODOLIST
React-TODOLIST just a reactjs.net example with with a few basic features.
### features:

  - CRUD TO DO LIST
  - Use quartz.net to set expired item 
  - Use webpack to module bundle
  - Use http://www.litedb.org/ for database app

### Remaining features
